Elastic Mechanics Demo

Welcome to the elastic mechanics HTML5 experiment. To draw a new shape, simply click on the screen to create new dots. With each new dot created, an elastic spring will attach to it. You may also connect two dots with additional elastic springs. This experiment uses Jquery and the RaphaelJS drawing library.

Note: This demo should be viewed only on Google Chrome.
